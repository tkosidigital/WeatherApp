User can add any city information by using two options, i.e. Manual or Select location on Map.Please go through the below steps to bookmark city.

  1. Please click on Plus Button ( + ) in right side corner.
  2. Select any one of the option ( Manual or Map ).
  3. Add City.

User can able to delete the added cities information by using the swipe and delete functionality. Once click on any city then application shows respected city weather information.
